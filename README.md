# FFXIV Adventurer's Scrapbook

旅人の机の上に記録を広げたような、スクラップブック風のFFXIV向けプロフィール画像を作成する非公式ファンツールです。

- 公開ページ：https://scaledwonderer-droid.github.io/Scrapbook/
- 元リポジトリ：https://github.com/scaledwonderer-droid/Scrapbook
- 初公開：2026年
- 企画・デザイン・制作：鱗の旅人

## 特徴

- 1920×1080px（16:9）のPNG画像を生成
- Aetheryte Pass、Traveler Memo、Inn Receipt、小写真2枚で構成
- キャラクターSSと小写真はドラッグによる位置調整、ズーム調整に対応
- 任意のQRコード画像を表示可能
- 外部ライブラリを使用せず、入力内容と画像はブラウザ内だけで処理
- アップロードした画像や入力内容をサーバーへ保存しません

## 使い方

1. キャラクターSSやプロフィール情報を入力します。
2. 必要に応じてQRコード画像、小写真、キャプションを設定します。
3. プレビュー上で画像の位置やズームを調整します。
4. 「PNG保存」から完成画像を保存します。

## ファイル構成

```text
README.md
index.html
scrapbook_base_back.png
```

`index.html` と `scrapbook_base_back.png` は同じ階層に配置してください。
